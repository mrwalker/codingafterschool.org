Coding for a Cause

Spicewood Coding Club

We will be taking ACTION in our community by donating all club proceeds to a local charity

Spicewood Coding Club is open to all grades K-5. We host three 8-week sessions, on Mondays after school. Signup is conducted in the fall through the Friday Folder.

Sponsors: Mrs. Haney, NextGen Teachers - Ms. Keating, Mrs. Pierce, Mrs. Martinez, Mrs. Dielmann, and parent volunteer, Matt Walker.

Coding Club is in its fourth year at Spicewood. It originated as an after school program for kindergartners using Code.org and has since expanded to include all grades and a variety of coding curricula. Coding Club is a fun way for kids to learn to program a computer.

Resources for learning to code

  • Code.org (Course 1)
  • Scratch (our studio)
  • ScratchJr (iPad app)
  • OzoBlockly
